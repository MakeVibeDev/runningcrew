## Vercel에서 설정하는 방법

1. Vercel 대시보드 → runningcrew-admin 프로젝트 선택
2. Settings → Environment Variables
3. 위의 환경 변수들을 하나씩 추가
   - Name: 변수명 (예: NEXT_PUBLIC_APP_TYPE)
   - Value: 값 (예: admin)
   - Environment: Production, Preview, Development 모두 선택
4. Save 버튼 클릭
5. 모든 변수 추가 후 Redeploy 실행

---

## 주의사항

⚠️ **NEXT_PUBLIC_APP_TYPE=admin** 이 변수가 가장 중요합니다!
   이 변수가 없으면 Admin 앱이 작동하지 않습니다.

⚠️ **NEXT_PUBLIC_SITE_URL** 값을 Admin 도메인으로 변경해야 합니다:
   - 로컬: http://localhost:3001
   - 프로덕션: https://admin.runningcrew.io (또는 실제 도메인)

⚠️ Kakao 로그인 관련 변수는 Main 앱에서만 사용되므로 Admin에는 불필요합니다.
